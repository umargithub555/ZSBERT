# 数据集介绍
  ## 简介
  FewRel（Few-Shot Relation Classification Dataset）包含来自 Wikipedia 的 100 个关系和 70,000 个实例。数据集分为三个子集：训练集（64 个关系）、验证集（16 个关系）和测试集（20 个关系）。
